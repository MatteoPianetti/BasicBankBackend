# Banking Backend - Spring Boot

Progetto backend bancario completo sviluppato con **Java Spring Boot**.  
Gestione sicura di **utenti, conti, carte e transazioni**, con controllo dei limiti e validazioni business. Testato con **Postman** e protetto con **Spring Security + JWT**.
